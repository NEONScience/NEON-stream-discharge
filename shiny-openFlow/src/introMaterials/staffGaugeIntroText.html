About the Gauge Height Calulcator

This tab calculates the gauge height from levelTROLLS/aquaTROLLS pressure data from the L0 database

How to use this tab:

  • Choose a domain and site.
  • Choose a water type; GW (ground water) or SW (surface water)
  • Choose a locaiton. This is site dependent. (see notes for examples)
  • Select the date range for the L0 pressure readings.
  • Select whether L0 data will be used or a single pressure reading

    • Important Notes:

  • If L0 selected is set to "Yes", there will be the option to upload a .csv file, the program will default to using it as long as it exists
  • When GW is selected as the water type there will be two input boxes for the well depths and troll cable length, both must be filled in otherwise they should be left blank. Leaving them blank will result in the water column height above the GWW troll excluding water depth underneath it. Additionally, GWW data results in water column height instead of a calculated gauge height.
  • L0 data formatting from the L0 data viewer tab of Blizzard is accepted in this program, Grafana is still a work in progress
